## Overview
This repository contains a Power BI dashboard designed to analyze sales performance, profitability, discount behavior, and product-level trends using interactive and drillable visuals.

The dashboard focuses on translating transactional sales data into clear, business-relevant insights through structured KPIs, comparative analysis, and time-based trend exploration.

---

## Business Objectives
The dashboard addresses the following analytical objectives:

- Identify top- and bottom-performing products by Sales, Profit, and Quantity Sold
- Analyze sales trends across multiple time granularities (daily, monthly, quarterly, yearly)
- Examine the relationship between Sales and Profit
- Compare Sales, Profit, and Quantity between two user-selected time periods
- Evaluate average discount behavior across promotion categories
- Track total number of orders
- Enable detailed, filterable transaction-level analysis
- Visualize sales distribution across cities

---

## Analytical Capabilities
- Interactive drill-up and drill-down on time-series visuals
- Period-over-period comparison using independent date filters
- KPI-driven overview for high-level analysis
- Controlled visual interactions to guide analytical focus
- Detailed tabular view for granular exploration

---

## Tools & Technologies
- Power BI Desktop
- Microsoft Excel (data source)

---

## Dataset Context
The dataset represents retail sales transactions including products, customers, promotions, pricing, discounts, profitability metrics, and city-level sales information.

Some periods contain partial data, which is intentionally retained to support realistic trend exploration during drill-down analysis.

---

## Usage
1. Download the `.pbix` file from this repository
2. Open it using Power BI Desktop
3. Use slicers, drill controls, and filters to explore insights at different levels of detail
